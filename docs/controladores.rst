Controladores
=============
Los controladores son el puente entre la vista y el modelo. Hay un archivo para 
cada sección: Conglomerado, Conteo de aves, Especies invasoras, Huellas y 
excretas, Vegetación y suelo, Epífitas, Impactos ambientales, Trampa cámara 
y Selección de fauna, Grabadora, Registros extra, Revisar registros y 
Exportar datos.
